## Задача 1
### Текст задачи
Первый массив, размерностью n х m , заполняется данными, вводимыми с клавиатуры, так что
заполнение ведется по строкам от первых элементов строки к последним.
Второй массив, размерностью n х n, заполняется так: элементы, лежащие выше побочной
диагонали, являются случайными числами из интервала [-65; 120], а лежащие на побочной
диагонали и ниже, являются случайными числами из интервала [-3.5; 10.75].
Третий массив, размерностью n х n, заполняется для произвольного n так же, как для n=5:
<img width="213" height="123" alt="image" src="https://github.com/user-attachments/assets/8edf0a30-da90-4d12-acc6-cbd0e144121d" />

### Алгоритм решения
Алгоритм 1.1: Заполнение массива с клавиатуры по строкам
Создать двумерный массив размером n×m
Организовать двойной цикл:
Внешний цикл по строкам (i от 0 до n-1)
Внутренний цикл по столбцам (j от 0 до m-1)
Для каждого элемента:
Считать значение с клавиатуры
Записать значение в массив data[i,j]

Алгоритм 1.2: Специальное заполнение относительно побочной диагонали
Создать квадратный массив n×n
Организовать двойной цикл по всем элементам
Для каждого элемента [i,j]:
Если элемент выше побочной диагонали (j < n-i-1):
Заполнить случайным целым числом из [-65; 120]
Если на побочной диагонали или ниже (j >= n-i-1):
Заполнить случайным дробным числом из [-3.5; 10.75]

Алгоритм 1.3: Заполнение треугольной матрицы
Создать квадратный массив n×n
Организовать двойной цикл по всем элементам
Для каждого элемента [i,j]:
Если выше главной диагонали (j > i): ввод с клавиатуры
Если на главной диагонали (j = i): ввод с клавиатуры
Если ниже главной диагонали (j < i): автоматически 0

## Задача 2
### Текст задачи
В городе П. есть m банков. Известны величины задолженностей банков друг другу. Укажите банк с
максимальным долгом

### Алгоритм решения
Получить количество банков из размера матрицы
Создать массив для хранения суммарных долгов каждого банка
Для каждого банка i:
Просуммировать все элементы i-й строки (долги банка i)
Проверить наличие ненулевых долгов
Найти банк с максимальной суммой долга
Вернуть индекс этого банка
